# 🐦 Cuckoo 餐饮作業系統 — 用戶指南 (v1.2.2)

歡迎使用 **Cuckoo (布谷鳥)**。這是一款專為中小型餐飲門店設計的「配方驅動型」作業系統。它不僅能處理點單，還能精確追蹤每一克原材料的去向。

---

## 🚀 快速安裝

### macOS 用戶
1. 打開 `Cuckoo_1.2.2_universal.dmg` 安裝包。
2. 將 **Cuckoo** 圖標拖入 **Applications (應用程序)** 文件夾。
3. **首次打開提示**：若提示「無法驗證開發者」，請在「系統設置 -> 隱私與安全性」中點擊「仍要打開」。

### Windows 用戶
1. 運行 `Cuckoo_1.2.2_x64_zh-CN.msi` 安裝程序。
2. 按照安裝嚮導點擊「下一步」直到完成。
3. 從桌面快捷方式啟動。

## 🏗 核心作業流程

### 1. 初始化：建立你的庫存 (Materials)
*   **路徑**：`基礎數據 -> 原材料`
*   **操作**：輸入所有原材料（如：五花肉、石螺、辣椒油）。
*   **提示**：建議為易耗品設置「最低庫存警告」。

### 2. 核心：編寫配方 (Recipes)
*   **路徑**：`基礎數據 -> 配方管理`
*   **亮點**：Cuckoo 支援「樹狀配方」。你可以創建一個「滷味底料」配方，然後在「滷豬蹄」配方中引用它作為半成品。
*   **防呆**：v1.2.2 增加了依賴保護，防止誤刪正在被其他產品使用的配方。

### 3. 上架：配置菜單 (Menu)
*   **路徑**：`銷售管理 -> 菜單配置`
*   **操作**：將配方關聯到菜單項，並設置售價。此時系統會自動根據配方成本計算出你的「毛利率」。

### 4. 進貨：採購與入庫 (Inventory)
*   **路徑**：`庫庫管理 -> 採購訂單`
*   **流程**：創建採購單 -> 到貨點收 -> 自動生成庫存批次 (FIFO)。

## 每日營運

### POS 點單
*   前端點單界面，支持加料、去料、規格選擇。
*   提交訂單後，後台會**自動預扣**相應原材料。

### KDS 廚房顯示
*   廚房工作站實時接收訂單。
*   點擊「完成」時，系統執行**真實庫存扣除**。

### 數據報表
*   實時查看銷售額、銷量排行及原材料消耗明細。

---

## 打印機配置
*   支援 **飛鵝雲打印機** (遠程) 與 **局域網打印機** (ESC/POS)。
*   在 `系統設置 -> 打印機` 中配置 SN 碼或 IP 地址。

## 📡 技術支持與備份
*   **本地優先**：數據存儲在您的電腦本地，無網絡也可營運。
*   **備份**：數據文件位於 `~/Library/Application Support/Cuckoo` (Mac) 或 `%APPDATA%/Cuckoo` (Win)。請定期備份 `cuckoo.db`。
*   **遠程協助**：v1.2.2 內置了心跳與自動報錯系統，若遇到異常，請聯繫技術支持並提供報錯截圖。
